# 關於這個 Blog

這是一個使用 VitePress 建置的個人技術 Blog，內容管理系統採用 GitHub Issues。

## 特色

- ✅ **簡約現代**：使用 VitePress 內建主題，注重閱讀體驗
- ✅ **內容管理**：透過 GitHub Issues 撰寫和管理文章
- ✅ **自動部署**：GitHub Actions 自動建置並部署至 GitHub Pages
- ✅ **讀者互動**：透過 GitHub Issues 讓讀者留言討論
- ✅ **SEO 優化**：自動生成 sitemap、meta tags 等

## 如何發佈文章

1. 在 [GitHub Repository](https://github.com/cscolabear/cscolabear.github.io) 建立新的 Issue
2. 使用 Markdown 格式撰寫內容
3. 添加 `blog` label
4. 完成後將 Issue 設為 `closed` 狀態
5. GitHub Actions 會自動將文章同步到網站

## 技術架構

- **前端框架**：VitePress
- **內容來源**：GitHub Issues API
- **部署平台**：GitHub Pages
- **自動化**：GitHub Actions
